commands for django

1) python -m venv myenv => to create virtual environment
cmd
2) myenv\Scripts\activate => to run the script for virtual environment

note: after this (myenv) is added to the starting of you terminal

3) pip install django => to install django

4) django-admin startproject myproject => to create project

5) cd myproject => change directory to myproject

6) python manage.py startapp myapp => to create app

7) python manage.py runserver => to run the server
